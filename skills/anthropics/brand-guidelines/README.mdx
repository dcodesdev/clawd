# Anthropic Brand Styling

## Overview

This skill provides access to Anthropic's corporate visual identity. It enables the application of official brand colors and typography to various artifacts and materials, ensuring consistent brand presentation across different outputs.

## Key Elements

### Color Palette

Anthropic's official brand colors include:

- Primary brand colors
- Secondary accent colors
- Neutral tones for backgrounds and text
- Color combinations for various design contexts

### Typography

Includes specifications for:

- Primary typeface recommendations
- Font weights and sizes
- Typography hierarchy
- Line spacing and alignment guidelines

### Design Standards

Covers:

- Logo usage and spacing
- Brand element placement
- Visual consistency guidelines
- Design principles for Anthropic materials

## Use Cases

Use this skill when creating:

- Documents that need Anthropic branding
- Visual artifacts with corporate styling
- Presentations or materials for Anthropic
- Any content benefiting from Anthropic's look-and-feel

This resource ensures all branded materials maintain visual consistency and align with Anthropic's established design standards.
