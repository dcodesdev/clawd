# Theme Factory

## Overview

The Theme Factory skill provides a comprehensive toolkit for styling various artifacts with professional designs. It offers both pre-configured themes and custom theme generation capabilities.

## Features

### Pre-set Themes

Access 10 professionally curated themes, each featuring:
- Carefully selected color palettes for visual harmony
- Font pairings optimized for readability and aesthetics
- Consistent styling across different artifact types

### Supported Artifacts

Style the following artifact types:
- Slides and presentations
- Documents and reports
- HTML pages
- Interactive components

### Custom Theme Generation

Generate custom themes on-demand for unique styling needs. Tailor colors, fonts, and visual properties to match specific brand requirements or design preferences.

## Use Cases

- Creating professionally styled presentations and reports
- Building branded web artifacts with consistent visual identity
- Applying multiple design themes to existing content
- Generating custom themes for specific projects or clients

## Integration

The Theme Factory skill works seamlessly with Claude artifacts, allowing you to apply styling directly to generated content.
