## Overview

The internal-comms skill is designed to assist with writing various types of internal communications using company-standard formats. This skill provides guidelines and templates for different communication types commonly used within organizations.

## Supported Communication Types

This skill helps you create:

- **3P Updates** - Third-party or partner updates
- **Company Newsletters** - Periodic company-wide communications
- **FAQ Responses** - Frequently asked question answers
- **Status Reports** - Project or initiative status updates
- **Leadership Updates** - Communications to leadership and executives
- **Project Updates** - Project milestone and progress updates
- **Incident Reports** - Internal incident documentation and communication

## How to Use

Identify the type of internal communication you need to write, then follow the corresponding guideline file and examples provided in the skill. Each communication type has specific formatting requirements and best practices to ensure consistency with your company's communication standards.
