## Overview

The pdf skill is a comprehensive toolkit for PDF manipulation and document processing. It provides guidance and patterns for working with PDF documents programmatically, including extracting content, creating new PDFs, and transforming existing documents.

## Capabilities

This skill covers:

- **Text Extraction** - Extract text content from PDF documents
- **Table Extraction** - Extract tabular data from PDFs
- **PDF Creation** - Create new PDF documents programmatically
- **Document Merging** - Combine multiple PDF files
- **Document Splitting** - Split PDF documents into separate files
- **Form Handling** - Work with PDF forms and fill form fields

## Use Cases

The pdf skill is useful for:

- Automated document processing workflows
- Data extraction from PDF reports and documents
- Generating PDF reports from application data
- Document management and organization
- Form automation and data collection

Whether you need to extract information from PDFs or generate new documents, this comprehensive toolkit provides the guidance needed to work with PDF files effectively.
